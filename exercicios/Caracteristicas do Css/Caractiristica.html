Herança

Herança significa que o elemento herda um propriedade do seu Pai.
Geralmente as propriedades que são herdadas dizem respeito ao TEXTO.

Quando não é especificado nada em uma div ela vai obter o estilo da herança da qual faz parte;

Especificidade

Quanto mais especificado o elemento no CSS for, maiores as chances dele ter seus estilos aplicados.

Como especificar uma classe em um css? Usando o ponto + o nome da classe. Ex: .container

cascata

Atribuições CSS INLINE são de execução prioritária, porém, é uma prática incorreta, pois dificulta a manutenção do código, suja o mesmo e não permite reuso de classes CSS.

Qual o grau de importância dos elementos inline no CSS?

Ele é o grau prioritário para alterações de estilo no CSS.

unidade de medida ch

Viewport

O que é um viewport?

Toda a área da página visível, a qual o usuário pode ver e usar.

A medida vh equivale-se a uma altura que seria 1/100 do(a) _viewport. unidade de media do espaço disponivel do meu usuário.
vh referente a altura disponivel
vw vai pegar a largura disponivel.
vmin vai usar o que for menor.
vmax vai usar o que for maior.

valor inherit

Force uma herança.
color:inherit;
background-color:inherit;

O inherit obriga a ter uma herançaentre os elementos. Isso ocorre principalmente com fontes
