Fazer lista no html


no ol colocamos um atributo chamado type:

Com o atributoSTART,podemos indicar de onde começará nossa lista.

Listas ordenada

:
          1. Café
          2. Chá
          3. Leite

          Listas desordenadas

          ;
          • Café
          • Chá
          • Leite


          Podemos criar uma lista dentro de um item que já era lista.
