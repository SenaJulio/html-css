EXISTEM 3 FORMAS DE INSERÇÃO DO CSS NO HTML.

1-Inline- colocado dentro do próprio Elemento.

2-Interno- colocar o codigo css em um unico dentro da pagina, recomentado dentro do Head.. criando um tag style.

3-Externo - criar um novo arquivo fora da minha pagina chamado de stylo.css

O que é e para que serve css.

O css é a forma visual do html, mudança na estrutura do html.

responsável pela a estilização da página, como alterar tamanho da fonte ou cor de fundo.

CSS é o estilo aplicado na estrutura ou item dos elementos do seu site;

para estilizar no Elemento sempre adicione a tag style dentro.

Base gral do css



id= identifica apenas um unico elemento. seleção do id= # + tag

class = . + tag pode colocar a class em qualquer elemento.

nome do elemento = pode selecionar tambem para a estilização..

Para unir as propriedades de id e de class= separa-los por virgulas
