Layout adaptativo vs layout responsivo.

layout adaptativo= esse layout foi pensando para ir diminuindo e se adapitando de acordo com o tamanho da tela.

layout responsivo= Página que se adapta a qualquer tamanho de dispositivo. Ou seja, os elementos serão organizados do menor ao maior tamanho;

Para que é usado o item @media no CSS ?

Para alterar as propriedades dos elementos de acordo com a resolução, na qual, a página está. @media only print , screen{ somente para quando for imprir... }

O que deve ser definido no @media para que as alterações sejam realizadas na página?

O tamanho da resolução mínima e máxima para realizar a alteração;

Funções min e max

1.Auxilia na responsividade da aplicação, estabelecendo limites de largura e altura no conteúdo.
2.Ela pode controlar o tamanho de imagens e fontes, como elas irão aparecer.
3.Ela pode controlar o tamanho de um input em um formulário.

width:max(50%, 300px)
width:min(50%, 300px)

função calc

A função calc no CSS permite que sejam feitos:

Cálculos que envolvam a altura e a largura de determinado item da aplicação.
Calcula os espaços exatos nos elementos para aplicar na página e deixar padrão e responsiva; width:calc(100%-20px)
